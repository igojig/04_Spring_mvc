
### Запуск
1. Все в докере:

    - `docker compose up -d`
    - `http://localhost:80`

2. Postgres в докере, старт через IDEA

    - `docker compose  -f .\postgres-compose.yaml up -d`
    - `запустить через IDEA приложение`
    - `http://localhost:8080`
